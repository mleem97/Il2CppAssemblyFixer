# 🛠️ Unity MelonLoader Il2CppAssemblyFixer

**Il2CppAssemblyFixer** is a specialized toolkit designed to repair corrupted IL2CPP assembly metadata and resolve reference issues in **Unity 6** environments using **MelonLoader** or **BepInEx**.

***

## ⚠️ The Problem: Update-Induced Corruption
Certain Unity updates (e.g., transitioning from `6000.3.x` to `6000.4.x`) trigger a bug in MelonLoader's assembly generation process. 

This primarily affects `UnityEngine.CoreModule.dll`, where the generator creates malformed managed DLLs with duplicate type definitions, causing the .NET runtime to crash on startup:

```text
System.BadImageFormatException: Duplicate type with name '<>O' in assembly 'UnityEngine.CoreModule'
```

### Known Unity 6 Breaking Changes
| Issue | Symptom | Root Cause |
|:--- |:--- |:--- |
| **CoreModule Crash** | `BadImageFormatException` | Duplicate `<>O` delegate cache types. |
| **Scene Crash** | `SceneHandler.Init()` crash | `Scene.GetNameInternal` requires `SceneHandle`. |
| **Stripping** | `Method Unstripping Failed` | `SceneManager.GetAllScenes()` is stripped. |
| **GC Issues** | `ObjectCollectedException` | premature `Il2CppObject` collection. |

***

## 🚀 Quick-Start
*Ensure you are using **MelonLoader v0.7.2** or higher.*

1. **Generate Assemblies:** Launch the game once so MelonLoader can generate the `Il2CppAssemblies` folder.
2. **Download Suite:** Download the latest [Release ZIP](../../releases).
3. **Apply Fixes:**
    *   **Standalone:** Run `Il2CppAssemblyFixer.exe` to auto-detect your game path and patch critical DLLs.
    *   **Persistent:** Extract `Il2CppAssemblyFixerPlugin.zip` to `<GameFolder>/Plugins/` to auto-patch after game updates.
    *   **UI Patch:** Move `UnityExplorerUnity6Shim.dll` to `<GameFolder>/Mods/` to resolve UnityExplorer crashes.

***

## 🧩 Why Use the Plugin?
Because the `BadImageFormatException` occurs during the earliest stages of game initialization, standard mods load too late to intervene.
*   **Early Execution:** The plugin hooks into `OnPreInitialization`.
*   **Automatic Detection:** It identifies if assemblies have been regenerated by a game update.
*   **Pre-emptive Patching:** Duplicates are patched before the runtime attempts to load the corrupted metadata.

***

## 🔧 CLI & Advanced Usage
Use the standalone EXE for manual repairs or CI/CD pipelines.

```powershell
.\Il2CppAssemblyFixer.exe              # Auto-detect & fix
.\Il2CppAssemblyFixer.exe --all        # Process all non-protected DLLs
.\Il2CppAssemblyFixer.exe --restore    # Revert all changes from .bak files
```

***

## 🛡️ Safety & Backups
*   **Non-Destructive:** Every modified file is backed up with a `.bak` extension.
*   **Safety:** The tool ignores protected system files to prevent engine-level instability.
*   **When to re-run:** You only need to manually run the EXE if the game updates or if the `Il2CppAssemblies` folder is cleared. Otherwise, the plugin handles maintenance automatically.
